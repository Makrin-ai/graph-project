Простое CLI-приложение для работы с CSV-конфигурациями на Java.
Описание работы приложения:
Приложение читает конфигурационные параметры из CSV-файла и предоставляет интерфейс командной строки для их просмотра.
Функциональность приложения:
- Чтение конфигурации из CSV-файла
- Простой CLI-интерфейс
- Просмотр всех параметров конфигурации
- Обработка ошибок и валидация данных

После запуска доступны команды:
show - показать все параметры конфигурации
help - показать справку по командам
exit - выйти из приложения
