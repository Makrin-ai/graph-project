Простое CLI-приложение для работы с CSV-конфигурациями на Java.
Описание работы приложения:
Приложение читает конфигурационные параметры из CSV-файла и предоставляет интерфейс командной строки для их просмотра.
Функциональность приложения:
CLI-приложение, которое:
- Загружает конфигурацию из CSV-файла
- Подключается к Maven-репозиторию
- Извлекает информацию о прямых зависимостях пакета
- Отображает список зависимостей в консоли

Основные компоненты:
- Main - главный класс приложения
- readConfig() - чтение CSV-конфигурации
- readUrl() - загрузка данных из URL
- Регулярные выражения для парсинга XML

После запуска доступны команды:
src/main/resources/config.csv - показать все параметры конфигурации
dependencies - показать информацию о прямых зависимостях заданного пользователем
пакета, используя URL-адрес репозитория
